Es una aplicacion web que funciona como blog personal, donde se crea, edita o elimmina posts.
URL de la pagina del proyecto: https://roadmap.sh/projects/personal-blog 
El frontend no es lo mio, lo siento.

# Requisitos

- Habra una Hompage que listara todos los articulos del blog, podra ser accedida por cualquiera.
- Habra una Detailpage que mostrara el detalle del articulo que se quiera, podra ser accedida por cualquiera.
- Habra una seccion para el admin a la que se ingresa con User: admin, Password: 1411.
- En la seccion para el admin se podra ver todos los articulos publicados, editar o eliminar el que se quiera.

# Instalacion
Para usar la aplicacion, seguir los siuientes pasos:
1. Clonar el repositorio:

        git clone https://github.com/chirogod/BlogPersonal.git 

2. Navegar a la raiz del proyecto

		cd BlogPersonal

3. Correr los comandos deseados: (cada comando debe tener dotnet run detras.)

		dotnet run

3. La consola dara una url a la que hay que acceder. Ejemplo

		http://localhost:5289
